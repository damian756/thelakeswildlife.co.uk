# The Lakes Wildlife (thelakeswildlife.co.uk)

**A wildlife database covering the birds, mammals, insects and plants of the Lake District.**

Live site: [thelakeswildlife.co.uk](https://www.thelakeswildlife.co.uk)

---

## What This Is

The Lakes Wildlife is an editorial wildlife guide to the Lake District. It covers species across four taxonomic groups, plus dedicated guides to the region's major habitats and reserves.

Built for birdwatchers, naturalists, wildlife photographers, and anyone interested in the ecology of the Lake District.

Part of [The Lakes Network](https://thelakes.network), built and operated by Churchtown Media.

## Stack

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Deployment**: Vercel
- **Analytics**: Plausible

## Key Features

- Species database across birds, mammals, insects and plants
- Client-side Wikipedia image fetching (avoids build-time API failures)
- Deterministic Wikipedia `Special:FilePath` URLs for OpenGraph images
- Habitat guides: Haweswater RSPB, Grizedale Forest, Martindale, and more
- Full Schema.org JSON-LD (Taxon, Article, TouristAttraction, FAQPage, BreadcrumbList)
- Species pages with conservation status, seasonal patterns, identification notes, photography tips
- Lightbox image viewer per species
- Sitemap covering all species and reserve pages

## Habitats Covered

- Haweswater RSPB (ospreys)
- Grizedale Forest (red squirrels, red deer)
- Martindale (red deer rut)
- Lake District uplands and tarns
